# Agent Registry

This repository is the **agent registry**, containing the definitions for agents, commands, prompts, and skills.

## Structure

- `agents/` — Agent definitions (e.g. backend-designer, frontend-designer, market-analyzer, researcher, software-engineer, etc.)
- `skills/` — Skill definitions organized by domain (e.g. backend, frontend, code-security, data-design, research, software-engineering)
